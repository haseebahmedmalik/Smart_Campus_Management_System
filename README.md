# 🎓 Smart Campus Management System

A console-based **Smart Campus Management System** developed in **C++** using **Object-Oriented Programming (OOP)** concepts. This application helps manage students, teachers, departments, courses, enrollments, and academic records through a user-friendly menu-driven interface.


## ✨ Features

- 👨‍🎓 Student Management
- 👨‍🏫 Teacher Management
- 🏢 Department Management
- 📚 Course Management
- 🏫 Section Management
- 📝 Enrollment Management
- 📊 Result & Grade Management
- 📋 Menu-Driven Interface
- 💾 File Handling for Data Storage


## 🛠️ Built With

- **Language:** C++
- **Programming Paradigm:** Object-Oriented Programming (OOP)
- **Compiler:** g++
- **IDE:** Visual Studio Code / Code::Blocks / Dev-C++


## 📚 OOP Concepts Used

- Classes & Objects
- Inheritance
- Encapsulation
- Polymorphism
- Templates
- Operator Overloading
- STL (Vectors)
- File Handling


## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/haseebahmedmalik/Smart_Campus_Management_System.git
```

### Compile

```bash
g++ *.cpp -o SmartCampus
```

### Run

```bash
./SmartCampus
```

> **Windows (MinGW):**
>
> ```bash
> SmartCampus.exe
> ```
